<code>
如果你希望函数不要再被派生类进一步重写，你可以把它标识为final。可以在基类或任何派生类中使用final。在派生类中，可以同时使用override和final标识。   
</code>cpp <br/>
class B
{
public:
   virtual void f(int) {std::cout &lt;&lt; &ldquo;B::f&rdquo; &lt;&lt; std::endl;}
};</p>

<p>class D : public B
{
public:
   virtual void f(int) override final {std::cout &lt;&lt; &ldquo;D::f&rdquo; &lt;&lt; std::endl;}
};</p>

<p>class F : public D
{
public:
   virtual void f(int) override {std::cout &lt;&lt; &ldquo;F::f&rdquo; &lt;&lt; std::endl;}
};
