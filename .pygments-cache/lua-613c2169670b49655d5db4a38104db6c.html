<code>  
可以看到在表的最后生成一个新表，只将需要的方法暴露给外边，像上边newAccount中的world()方法就只能在内部访问，外部无法访问。   
[《Programming in Lua》](http://www.lua.org/pil/16.5.html)中讲到一种使用闭包实现更高效的方法。   
</code>lua
function newObject( value )
    return function (action, v)
        if action == &ldquo;get&rdquo; then return value
        elseif action == &ldquo;set&rdquo; then value = v
        else error(&ldquo;invalid action&rdquo;)
        end
    end
end</p>

<p>d = newObject(0)
print(d(&ldquo;get&rdquo;))    &ndash;> 0
d(&ldquo;set&rdquo;, 10)
print(d(&ldquo;get&rdquo;))    &ndash;> 10
