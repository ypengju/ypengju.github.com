class B
{
public:
   virtual void f(short) {std::cout &lt;&lt; &ldquo;B::f&rdquo; &lt;&lt; std::endl;}
};</p>

<p>class D : public B
{
public:
   virtual void f(int) {std::cout &lt;&lt; &ldquo;D::f&rdquo; &lt;&lt; std::endl;}
};
<code>  
D::f 按理应当重写 B::f。然而二者的声明是不同的，一个参数是short，另一个是int。因此D::f（原文为B::f，可能是作者笔误——译者注）只是拥有同样名字的另一个函数（重载）而不是重写。当你通过B类型的指针调用f()可能会期望打印出D::f，但实际上则会打出 B::f 。   
另一个很微妙的错误情况：参数相同，但是基类的函数是const的，派生类的函数却不是。
</code>cpp<br/>
class B
{
public:
   virtual void f(int) const {std::cout &lt;&lt; &ldquo;B::f &rdquo; &lt;&lt; std::endl;}
};</p>
